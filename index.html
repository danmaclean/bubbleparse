Welcome

Details of installation and use can be found in manual.pdf.

A tutorial is provided in tutorial.pdf, which guides you through finding the SNPs between two accessions of Arabidopsis thaliana, reads for which are available in the EBI Sequence Read Archive.

The code is designed to be used with Cortex Con (not Cortex Var), which is available from: https://sourceforge.net/projects/cortexassembler/files/cortex_con/
